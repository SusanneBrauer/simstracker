Add a Generic In Game Household

This option will allow you to add one of the pre-made in game households to your tracker. It will add the original Sims in the state as they are when you are creating a new fresh safe. You will still be able to modify them if they have already changed from that in your save but it wll safe creating Sims who are essentially known. Plumtree family trees will not be included, but if you want to create one you can still link it.




Add a Custom Household

This option will allow you to add your own households to your tracker. It will add an empty household you will add Sims and other properties to it in the next step. Plumtree family can be linked.
